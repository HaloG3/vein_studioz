# Vein Studioz Website

A static portfolio website for Vein Studioz, built with HTML, CSS, and JavaScript.

## Files

- `index.html` - landing page markup with SEO metadata and portfolio sections.
- `style.css` - responsive styling, animation effects, and Figma-inspired design.
- `script.js` - mobile menu toggle and scroll reveal animations.

## How to use

1. Open `index.html` in your browser.
2. Replace placeholder project images with your own collaboration photos.
3. Update contact details, social links, and YouTube channel links.
4. Optionally host on GitHub Pages, Netlify, or Vercel for a shareable portfolio website.

## Notes

- The page is styled to look modern, animated, and social-ready.
- The layout is SEO-friendly with meta description, keywords, and open graph metadata.
